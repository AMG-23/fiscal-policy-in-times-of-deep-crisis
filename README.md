# Fiscal Policy in Times of Deep Crisis

This repository contains the replication code for the three papers included in the PhD dissertation *Fiscal Policy in Times of Deep Crisis*.

## Contents

- Paper 1 – Fiscal Policy during a Pandemic: The Mechanisms behind the Aggregate Effects of a Temporary VAT Cut
- Paper 2 – The German Energy Crisis: A TENK-Based Fiscal Policy Analysis
- Paper 3 – Transition Dynamics in Heterogeneous-Agent Models and the Distributional Consequences of Taxation

Please refer to the README file in each paper folder for detailed replication instructions.
